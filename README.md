# Projeto Login - Git e GitHub

## Descrição
Este projeto foi desenvolvido como atividade prática de Git e GitHub.

A aplicação consiste em uma tela de login com tema escuro, validação de campos em JavaScript e páginas adicionais para dashboard e cadastro de usuário.

## Funcionalidades
- Tela de login
- Validação de campos com JavaScript
- Mensagem de erro para campos vazios
- Mensagem de sucesso para login válido
- Dashboard simples com menu e mensagem de boas-vindas
- Tela de cadastro de usuário

## Estrutura do Projeto
- `index.html` → página principal de login
- `style.css` → estilos da aplicação
- `script.js` → validação do formulário de login
- `dashboard.html` → painel principal
- `cadastro.html` → tela de cadastro
- `README.md` → documentação do projeto

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Fluxo de Versionamento
O projeto foi organizado com as seguintes branches:

- `main` → versão principal e estável
- `develop` → desenvolvimento do projeto
- `feature/validacao` → validação do login
- `feature/dashboard` → criação do painel principal
- `feature/cadastro-usuario` → criação da tela de cadastro
- `hotfix/erro-html` → correção de erro estrutural no HTML
- `release/v1.1.0` → preparação da versão final para publicação
